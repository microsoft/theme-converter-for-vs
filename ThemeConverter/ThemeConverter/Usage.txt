ThemeConverter v0.0.2
A utility that converts VSCode theme json file(s) to VS pkgdef file(s).
For Microsoft internal usage only.

Args:
	-patch <source_file/folder_path> <vs_install_dir>
			Convert the theme json(s) in the source path and patch
			the generated pkgdef(s) to the target VS, run /updateConfiguration
			and launch the target VS.
	-convert <source_file/folder_path> optional:<output_path>
			Convert the theme json(s) in the source path and output 
			the converted pkgdef(s) to the target path (will output to
			the source path if not specified)
	-help   Show help text.

Example:
	ThemeConverter -patch  "C:\foo\BestTheme.json" "C:\Program Files\Microsoft Visual Studio\2022\Preview"
	ThemeConverter -convert "C:\foo\ThemeFolder" "C:\ConvertResult"
	ThemeConverter -convert "C:\foo\BestTheme.json" "C:\ConvertResult"
